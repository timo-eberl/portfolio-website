About the Mesh Slicing Tool

This tool has been created to be used in a VR game where you can slice fruits thrown at you by other players. The game is still in development (as of 2023-11-28).

It is implemented as a Extension for the Godot Engine (GDExtension) and adds the Node SliceableMeshInstance3D that can slice a mesh along a plane.
